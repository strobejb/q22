Folder-box SVG icon set
=======================

Each color folder contains:
- single-closed.svg   original lighter single box
- single-open.svg     darkened single box
- double-closed.svg   closed double box
- double-open.svg     dark-front / subtle-seam double box

Colors:
- blue
- pink
- gray       (mid-gray)
- green      (emerald green)
- gold

All icons:
- transparent backgrounds
- 64×64 viewBox
